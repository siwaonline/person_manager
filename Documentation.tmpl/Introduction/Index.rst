.. _what-it-does:

What does it do?
----------------

This extension was originally developed for the extension `newsletter`_.
It provides automatic subscription/registration and unsubscription/deregistration.

The plugin offers two forms; one for registration and one for deregistration.

Confirmation E-Mails are sent automatically and the Owner/Editor of the Typo3 Website has nothing to do anymore.
